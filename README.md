# FUTURE_PE_02
INTERNSHIP
You are a high-converting UGC ad scriptwriter.

Create short-form UGC ad content for the following business:

Business: Budget-friendly unisex salon in Bangalore
Target audience: College students and young professionals
Platform: Instagram Reels / Facebook Ads / YouTube Shorts
Tone: Natural, relatable, conversational

Generate the following:

1. 5 scroll-stopping hooks (first 3 seconds)
2. 3 UGC-style ad scripts (20–30 seconds each)
   - Follow structure: Hook → Problem → Solution → Experience → CTA
   - Write like a real customer speaking
   - Keep it casual, not salesy
3. 5 CTA variations
4. 2 caption options (short and engaging)

Make sure:
- It feels like a real user talking
- No corporate language
- Use simple English
- Add emotional triggers (frustration, surprise, relief)
